# BAC Datathon - Preliminary Round

This project contains the analysis and data for Theme 2 of the BAC Datathon.

## 📂 Files Included
- `theme2_eda (1).ipynb`: Python Notebook for Exploratory Data Analysis.
- `Datathon.R`: R script for statistical modeling.
- `Theme 2 Cleaned.csv`: The processed dataset used for analysis.
- `BAC Datathon Preliminary Round Handbook.pdf`: Project guidelines and requirements.

## Note: Please run the Python file first, before running the R file.
